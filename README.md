#### Mappings
Due to limitations of MacOS I added new shortcut keys for window resizing


|Action   |Mappings          |
|---------|------------------| 
|`<C-a>`  |Resize split up   |
|`<C-z>`  |Resize split down |
|`<C-x>`  |Resize split left |
|`<C-c>`  |Resize split right|
